# work_init/

> 新規作品を起こすための bootstrap 手順。
> renji を実走した結果から抽象化された手順。

## ファイル

- `new_work_bootstrap.md` — step-by-step

## 使うタイミング

- 新規作品を始めるとき
- 既存資料を整理して新作品 directory に再構築するとき

## 使わないタイミング

- 既存作品の継続作業（writing/ で個別 ep の Writing Pack 生成へ進む）
- proposals 系の議論セッション（議論は inbox/planning_sessions/ に直接保存）
