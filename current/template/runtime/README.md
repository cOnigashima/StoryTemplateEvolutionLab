# runtime — 作品側で埋まる空フォルダ雛形

ここは「作品を current からコピーした時に付いてくる、作業ログと成果物の空の器」。
current/(正本)の中では空。実走が始まると各作品フォルダの中で埋まる。

| フォルダ | 中身 | 人間ゲート |
|---|---|---|
| `drafts/` | 本文候補（draft workflow の出力） | G-Deliverable |
| `reviews/` | レビュー票（採否判定付き） | G-Deliverable |
| `approved/` | 承認済み本文 | G-Publish |
| `published/` | 公開済み | — |
| `reader-export/` | 公開用テキスト（製作メタ除去。将来のiPad→PC Viewer用） | — |
| `logs/runs/` | TAKT実行ログ | — |
| `logs/sessions/` | セッションダイジェスト | — |
| `logs/learning/` | retro・方法論（→ current/template 昇格の材料） | — |

「人間は成果物だけ見る」の“成果物”は主に drafts/ reviews/ approved/。
