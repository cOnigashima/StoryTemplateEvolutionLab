# kakuyomu-policy — カクヨム投稿の作品横断ルール

## 基本ルール

- 投稿可能な作品は「会員自らが創作した作品」に限る
- AI利用作品は推奨タグを付ける

### AI利用作品の推奨タグ（2025年11月規定時点）

| タグ名 | 利用条件 |
|--------|----------|
| AI本文利用 | 本文の全体または大部分をAIが生成した作品 |
| AI本文一部利用 | 本文の一部にAIを利用した作品 |
| AI補助利用 | 校正・推敲・アイデア出しなど、補助的にAIを利用した作品 |

**この工場で生成された原稿は原則「AI本文利用」に該当する**。タグ設定は人間ゲート（G-Publish）で確認。

## 投稿頻度

作品ごとに `work.manifest.json` の `kakuyomu.publish_cadence` に設定。推奨上限（2025-11 時点の目安）:

- 1日あたり最大3エピソード / エピソード間隔は最低4時間 / 週あたり最大15エピソード
- 「通常の創作活動では考えられない短期間の膨大投稿」は控える

## 公開単位・出力

- **公開単位**: Episode = 公開単位。overlay の刻みに関わらず、公開は ep 番号で行う。
- **reader-export**: 公開用テキストは製作メタ（レビュー票・状態台帳・伏線メモ）を除去して `runtime/reader-export/` に出す。

## 禁止コンテンツ

過度な暴力描写 / 性的描写の規約違反 / 著作権侵害 / 第三者の権利侵害 / 差別的表現。

## コンテスト応募時の注意

- AI利用の可否・条件はコンテストごとに異なる。**応募前に必ず応募要項を確認**。
- カクヨムコン: AI利用作品は推奨タグの付与が必須。ジャンルによって追加制限あり。

## 公開前チェック（G-Publish の判断材料）

1. AI利用タグが正しく設定されているか
2. 投稿頻度が制限内か
3. 禁止コンテンツが含まれていないか
4. コンテスト条件に適合しているか（応募時）

## 緊急停止条件

規約違反の疑い / 警告・指導を受けた / 投稿頻度が制限を超えそう → 投稿を停止し author 判断。

## 法務

生成過程のログ（編集・構成・表現選択）を `runtime/logs/` に残し説明可能にする。類似性＋依拠性による侵害に注意。

## 参考（最新規約の確認先）

カクヨム利用規約 / 投稿ガイドライン / AI生成作品に関するお知らせ（2025年11月）— 規約は変わり得るので公開前に毎回チェック。
